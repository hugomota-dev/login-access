# Web Base - Login Acess

### Tipo: Projeto de estudo - Front-end

#### Objetivo Geral:
Praticar HTML5, CSS3 e JavaScript Vanillha por meio da construção de uma aplicação simples de login e gerenciamento de usuários, aplicando boas prática de desenvolvimento e qualidade de software.

##### Escopo do projeto
Tela de login
Cadastro de usuário
Listagem de usuário
Edição de usuário
Exclusão de usuários
Persistência de dados
